How it works

CUE.ai runs a background folder watch daemon on your Mac. It continuously monitors the folder synced from your phone. When a new voice memo is saved, it instantly copies, uploads to Gemini 2.5 Flash, transcribes, and dynamically adds the new insights directly to your dashboard.

Mobile Setup (iPhone or Android)

  1. Configure Shared Cloud Folder:

    On your phone, set up a voice recorder application (e.g. Apple Voice Memos, Voice Recorder Pro, or standard cloud app) to save or upload voice recorded files directly to a folder in your iCloud Drive, Google Drive, or Dropbox.

  2. Configure Path in CUE:

    Add the local path to this folder on your Mac in your CUE.ai .env file under the CLOUD_RECORDINGS_DIR key.

  3. Run Watch Daemon:

    Open a new terminal tab in the CUE directory and run: npm run sync. This launches the real-time observer daemon.
