📊 Google Sheets to IndexedDB Sync Guide

📋 What you'll learn: How to sync data from Google Sheets to IndexedDB using the built-in sync system, including setup, configuration, and practical examples.

🔍 Overview

Your application includes a comprehensive sync system that can automatically pull data from Google Sheets and store it in IndexedDB for offline access and improved performance. This guide explains how to use this system effectively.

Key Components

  • SheetsToIndexedDBSync: Core sync service that handles data transfer
  • IndexedDBManager: Manages local database operations
  • SyncSystemIntegration: Coordinates all sync components
  • Auto-Configuration: Automatically detects API keys and sheet IDs

⚙️ Setup & Configuration

Step 1: Configure Google Sheets API

First, ensure you have your Google Sheets API key and Sheet ID configured:

// Option 1: Environment Variables (Recommended for production) GOOGLE_SHEETS_API_KEY=your_api_key_here GOOGLE_SHEETS_SHEET_ID=your_sheet_id_here // Option 2: Use the configuration interface // Visit: /google-sheets-config-interface.html

Step 2: Include Required Scripts

Make sure these scripts are loaded in your HTML:

<script src="js/google-sheets-auto-config.js"></script> <script src="js/indexeddb-manager.js"></script> <script src="js/sheets-to-indexeddb-sync.js"></script> <script src="js/sync-system-integration.js"></script>

Step 3: Prepare Your Google Sheet

Structure your Google Sheet with the following tabs and headers:

  • Products: ID, Name, Category, PriceList, Price, Description, Active
  • Clients: ID, Name, Email, Phone, Address, Salesperson, PriceList, Active
  • Salespeople: ID, Name, Email, Phone, Territory, Active
  • PriceLists: ID, Name, Description, Active, EffectiveDate

🔧 Sync Methods

Method 1: Complete System Initialization

Initialize the entire sync system with automatic configuration:

// Initialize the complete sync system const syncSystem = new SyncSystemIntegration({ // Configuration will be auto-detected from environment or localStorage syncInterval: 300000, // 5 minutes logLevel: 'info', enableStatusUI: true }); // Initialize and start syncing await syncSystem.initialize(); // Perform initial sync from Google Sheets to IndexedDB await syncSystem.performFullSync();
Method 2: Direct Sync Service Usage

Use the sync service directly for more control:

// Initialize IndexedDB Manager const dbManager = new IndexedDBManager({ dbName: 'INH_SyncDatabase', logLevel: 'info' }); await dbManager.initialize(); // Initialize Sheets to IndexedDB sync service const sheetsSync = new SheetsToIndexedDBSync({ apiKey: googleSheetsAutoConfig.getApiKey(), sheetId: googleSheetsAutoConfig.getSheetId(), dbManager: dbManager, logLevel: 'info' }); // Sync all collections const results = await sheetsSync.syncAll(); console.log('Sync results:', results);
Method 3: Sync Specific Collections

Sync only specific data types:

// Sync only products const productResult = await sheetsSync.syncCollection('products'); // Sync only clients const clientResult = await sheetsSync.syncCollection('clients'); // Sync multiple specific collections const collections = ['products', 'clients', 'salespeople']; for (const collection of collections) { try { const result = await sheetsSync.syncCollection(collection); console.log(`${collection} sync result:`, result); } catch (error) { } }

Example 2: Sync with Progress Tracking

async function syncWithProgress() { const dbManager = new IndexedDBManager(); await dbManager.initialize(); const sheetsSync = new SheetsToIndexedDBSync({ apiKey: googleSheetsAutoConfig.getApiKey(), sheetId: googleSheetsAutoConfig.getSheetId(), dbManager: dbManager }); const collections = ['products', 'clients', 'salespeople', 'priceLists']; const totalCollections = collections.length; for (let i = 0; i < collections.length; i++) { const collection = collections[i]; const progress = Math.round(((i + 1) / totalCollections) * 100); console.log(`📊 Syncing ${collection}... (${progress}%)`); try { const result = await sheetsSync.syncCollection(collection); console.log(`✅ ${collection}: ${result.recordCount} records synced`); } catch (error) { } } console.log('🎉 Sync process completed!'); }

Example 3: Scheduled Automatic Sync

class AutoSyncManager { constructor() { this.syncInterval = null; this.isRunning = false; } async startAutoSync(intervalMinutes = 5) { if (this.isRunning) { console.log('Auto sync already running'); return; } // Initialize sync system this.syncSystem = new SyncSystemIntegration(); await this.syncSystem.initialize(); // Start periodic sync this.syncInterval = setInterval(async () => { try { console.log('🔄 Starting scheduled sync...'); await this.syncSystem.sheetsToIndexedDBSync.syncAll(); console.log('✅ Scheduled sync completed'); } catch (error) { } }, intervalMinutes * 60 * 1000); this.isRunning = true; console.log(`🚀 Auto sync started (every ${intervalMinutes} minutes)`); } stopAutoSync() { if (this.syncInterval) { clearInterval(this.syncInterval); this.syncInterval = null; this.isRunning = false; console.log('⏹️ Auto sync stopped'); } } } // Usage const autoSync = new AutoSyncManager(); autoSync.startAutoSync(5); // Sync every 5 minutes

Example 4: Sync with Error Handling and Retry

async function robustSync(maxRetries = 3) { let attempt = 0; while (attempt < maxRetries) { try { attempt++; console.log(`🔄 Sync attempt ${attempt}/${maxRetries}`); const syncSystem = new SyncSystemIntegration(); await syncSystem.initialize(); const result = await syncSystem.sheetsToIndexedDBSync.syncAll(); if (result.success) { console.log('✅ Sync successful on attempt', attempt); return result; } else { throw new Error('Sync returned failure status'); } } catch (error) { if (attempt >= maxRetries) { throw error; } // Wait before retry (exponential backoff) const delay = Math.pow(2, attempt) * 1000; console.log(`⏳ Waiting ${delay}ms before retry...`); await new Promise(resolve => setTimeout(resolve, delay)); } } }

🔧 Troubleshooting

Common Issues and Solutions

1. API Key or Sheet ID Not Found

Problem: "API key not configured" or "Sheet ID not found" errors
Solution:
  • Visit /google-sheets-config-interface.html to configure your settings
  • Or set environment variables: GOOGLE_SHEETS_API_KEY and GOOGLE_SHEETS_SHEET_ID
  • Check that your API key has the correct permissions

2. Google Sheets Access Denied

Problem: 403 Forbidden or access denied errors
Solution:
  • Ensure your Google Sheet is publicly readable or shared with the service account
  • Verify your API key is valid and has Google Sheets API enabled
  • Check the sheet ID is correct

3. Data Not Syncing

Problem: Sync completes but no data appears in IndexedDB
Solution:
  • Check that your sheet tabs have the correct names (Products, Clients, etc.)
  • Verify the first row contains the expected headers
  • Ensure required fields are not empty
  • Check browser console for validation errors

4. Performance Issues

Problem: Sync takes too long or times out
Solution:
  • Reduce the amount of data in your sheets
  • Sync specific collections instead of all at once
  • Increase timeout values in the sync configuration
  • Consider using incremental sync for large datasets

Debugging Tools

Use these tools to diagnose sync issues:

  • IndexedDB Diagnostic: /indexeddb-diagnostic.html
  • Google Sheets Config: /google-sheets-config-interface.html
  • Admin Sync Panel: /admin-sync-panel.html
  • Browser DevTools: Check Console and Application tabs
